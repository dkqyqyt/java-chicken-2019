# java-chicken-2019  
1. 주문을 등록  
    - 테이블 목록을 출력한다.
    - 등록할 테이블을 고를 수 있다.  
    [예외] 존재하지 않는 테이블 번호를 고를 경우
    - 메뉴 전체를 출력한다.
    - 메뉴를 고를 수 있다.  
    [예외] 존재하지 않는 메뉴 번호를 고를 경우
    - 주문할 메뉴의 수량을 선택할 수 있다.  
    [예외] 한 테이블에서 주문할 수 있는 갯수는 99개 이하이다.
    - 주문이 등록된 테이블은 결제 전까지 별도 표시한다.
    
2. 결제
    - 치킨 메뉴 수량 합이 10개 이상일 경우 10000원 씩 할인한다. (10 개 단위)
    - 현금 결제 시 5% 할인한다. (할인 중복 가능)
    - 결제 불가 시 이유를 보여준다.
    - 최종 결제 금액을 보여준다.
